## 1. 우리가 만든 게 뭐야?

**현대·기아 전기차 배터리 자가진단 웹앱**

지금 전기차 계기판에는 SOH(배터리 잔존 수명, %)만 숫자로 표시됨.
우리 앱은 여기서 더 나아가:

- **지금 내 배터리가 얼마나 남았는지** (현재 SOH 추정)
- **앞으로 몇 년 더 쓸 수 있는지** (미래 열화 곡선 예측)
- **언제 교체해야 하는지** ("3년 2개월 후")
- **어떻게 하면 더 오래 쓸 수 있는지** ("급속충전 줄이면 +1년 5개월")

한 줄 요약: **"내 아이오닉5 몇 년 더 버티냐"를 AI가 계산해주는 앱**

## 2. 왜 이 프로젝트를 선택했나?

- 전기차 배터리 교체 비용이 **1,200만원** — 소비자 입장에서 가장 큰 관심사
- 기존 앱/서비스는 현재 SOH만 보여주고 미래 예측 없음
- NCM 배터리(현대·기아 EV의 99%)는 열화 패턴이 데이터로 예측 가능
- 실차 데이터(191대)가 공개되어 있어 실제 적용 가능한 수준

## 3. 데이터셋: EVBattery가 뭐야?

**출처:** Figshare 공개 데이터셋 "EVBattery: A Large-Scale Electric Vehicle Dataset"
**원본:** 중국 실제 운행 전기차에서 수집한 충전 데이터

### 데이터 구조
- **총 465대** 전기차 데이터 → 품질 필터 후 **191대** 사용
  - 학습: 153대 (85,648 충전 세션)
  - 테스트: 38대 (21,629 충전 세션)

- **1건 = 충전 세션 1번** (충전기 꽂은 순간부터 완료까지)
- 각 세션에 포함된 정보:
  - 전압 (3.73~4.14V 범위 측정값들의 평균/표준편차)
  - 전류 (충전 속도)
  - 온도 (최대 53°C 기록)
  - 용량 (이번 충전으로 채운 kWh)
  - SOH (이 세션에서 측정된 배터리 잔존 용량 %)
  - 누적 주행거리, 사이클 수 등

### 왜 이 데이터?
- **NCM 화학** (아래 섹션 참고) — 현대·기아 EV와 동일
- 실차 운행 데이터 — 실험실이 아닌 실제 도로 조건
- 465대로 논문급 규모

## 4. NCM이 뭐야? 왜 중요해?

배터리는 **양극재 화학**에 따라 열화 패턴이 완전히 달라짐.

| 화학 | 풀네임 | 특징 | 누가 쓰나 |
|------|--------|------|-----------|
| **NCM** | 리튬 니켈 코발트 망간 산화물 | 에너지 밀도 높음, 열화 패턴 특정적 | **현대·기아 EV 99%** |
| LFP | 리튬 인산철 | 안전하고 저렴, 느린 열화 | 테슬라 스탠다드, BYD |
| LCO | 리튬 코발트 산화물 | 오래된 방식, 소형 기기용 | **NASA 실험 배터리** |

**핵심:** NASA 데이터(LCO)로 학습하면 현대·기아(NCM)에 적용 불가.
화학이 다르면 열화 메커니즘 자체가 달라서 모델이 틀린 예측을 함.
→ **우리는 NCM 실차 데이터(EVBattery)를 선택한 것.**

## 6. 모델 4개를 왜 비교했나?

단일 모델보다 여러 모델을 비교하면 "방법론의 견고성"을 증명할 수 있음.
→ "우리 접근법이 특정 모델에만 운 좋게 잘 되는 게 아니다"

### LSTM (Long Short-Term Memory)
- **뭐냐:** 순서가 있는 데이터(시계열)를 잘 처리하는 딥러닝
- **왜 썼냐:** 배터리는 세션 순서가 중요함 — 어제 어떻게 충전했는지가 오늘 SOH에 영향
- **특징:** 멀티태스크 (SOH + 열화율 동시 예측), MC Dropout으로 불확실성 계산
- **결과: R²=0.9233 (1등)**

### Transformer
- **뭐냐:** 자연어 처리(ChatGPT의 기반)에서 쓰이는 Attention 구조를 시계열에 적용
- **왜 썼냐:** 30개 충전 세션 중 "어떤 세션이 현재 SOH에 중요한지" 자동으로 가중치 부여
- **결과: R²=0.9169 (2등)**

### XGBoost
- **뭐냐:** 의사결정나무를 여러 개 합친 기계학습 (딥러닝 아님)
- **왜 썼냐:** 딥러닝 없이도 잘 되는지 비교하기 위해 (baseline)
- **특징:** 30개 세션의 통계값(평균, 표준편차, 최대/최소)을 피처로 사용
- **결과: R²=0.9084 (3등)**

### Neural CDE (Neural Controlled Differential Equation)
- **뭐냐:** Kidger et al. 2020 (NeurIPS)에서 발표한 불규칙 시계열 전용 모델
- **수식:** dZ(t) = f_θ(Z(t)) · dX(t) — 데이터를 연속 경로로 보고 미분방정식으로 적분
- **왜 썼냐:** 충전 세션 간격이 불규칙 → CDE가 이론적으로 더 적합
- **결과: R²=-49 (발산)** — 실제 데이터는 생각보다 규칙적이어서 LSTM이 더 잘 맞음
- **발표 포인트:** "이론적 우수성과 실제 성능은 다를 수 있다 → 실험 비교의 중요성"

## 7. Physics-Informed란 뭐야?

단순히 데이터만 학습하는 게 아니라, **배터리 물리 법칙을 손실함수에 내장**했다는 뜻.

### Peukert 법칙 (충전 속도 영향)
- **내용:** 빠르게 충전할수록(높은 C-rate) 배터리 수명이 단축됨
- **수식:** 수명 ∝ (1/C-rate)^n, n=0.621
- **근거:** Safari & Delacourt 2011, Pelletier et al. 2017
- **실생활:** 급속충전 자주 쓰면 수명 절반 가까이 줄어듦

### Arrhenius 법칙 (온도 영향)
- **내용:** 온도가 10°C 오를 때마다 열화 속도 ~1.5배 빨라짐
- **수식:** 열화율 ∝ exp(Ea/R·(1/T_ref - 1/T)), Ea=31,000 J/mol
- **근거:** Wang et al. 2014, Waldmann et al. 2014
- **실생활:** 여름 옥상 주차 vs 지하주차 → 수명 2~3배 차이

### PINN 손실함수
훈련할 때 "Peukert에 위반되는 예측은 패널티"를 추가로 줌.
→ 모델이 물리 법칙을 따르도록 제약 → 데이터 외 영역(SOH 80% 이하)에서도 신뢰할 수 있음

## 8. 배터리 열화는 왜 쓸수록 빨라지나? (2-phase 모델)

배터리 수명은 **두 단계**로 열화함. 이걸 코드에 구현했음.

### Phase 1: 초기 안정 열화 (SOH 100~85%)
- **원인:** SEI (Solid Electrolyte Interphase) 막 성장
  - 음극 표면에 전해질이 분해되어 얇은 막이 생김
  - 이 막이 자라면서 리튬이 갇혀 용량 손실
- **속도:** √t 비례 (초반엔 빠르게, 갈수록 느려짐)
- **근거:** Ploehn et al. 2004 (J. Electrochem. Soc.)

### Phase 2: 가속 열화 — 무릎 효과 (SOH < 85%)
- **원인1:** 리튬 도금 (Li Plating) — 고속 충전 시 음극에 리튬 금속이 석출
- **원인2:** 양극 구조 붕괴 — NCM 결정 구조가 무너지기 시작
- **결과:** 열화 속도가 지수함수적으로 증가 → 수명 곡선에 "무릎(Knee)" 생김
- **근거:** Attia et al. 2022 (Nature Energy), Cannarella & Arnold 2014

### 승법 노이즈 (불확실성)
- σ(t) = 1.2 × √(t/EOL)
- 쓸수록 예측 불확실성이 커짐 — 오래된 배터리는 예측하기 어려움
- 앱에서 SOH ± X% 로 표시

## 9. 앱에서 무슨 일이 일어나나? (흐름 설명)

사용자가 **🔍 분석하기** 버튼을 누르면:

```
1. 운전 습관 5개 입력 (속도, 급가속, 급제동, SOC폭, 충전빈도)
         ↓
2. Random Forest 분류기 → 절약형/평균형/공격형 판정
         ↓
3. 차종 spec(kWh, kW) + 운전유형 → 실효 C-rate 계산
         ↓
4. Peukert × Arrhenius → 이 사람의 EOL 사이클 수 계산
         ↓
5. 2-phase 열화 모델 → SOH 미래 곡선 생성 (2500 사이클)
         ↓
6. LSTM 앵커 → 현재 SOH 추정값으로 곡선 보정
         ↓
7. 맞춤 TOP 3 추천 출력 (효과 큰 것부터)
```

## 10. 발표할 때 자주 나오는 질문 대비

**Q: 중국 데이터인데 한국 EV에 써도 되나?**
A: NCM 화학이 동일하고, Peukert/Arrhenius 물리 법칙은 화학에 의존하므로 적용 가능합니다.

**Q: R² 0.92인데 왜 낮냐?**
A: R²는 "평균 대비 얼마나 더 잘 맞추냐"인데, SOH 분포가 95~100%에 몰려있어 분산이 작습니다. RMSE 0.78%가 실용적 기준이고, 이는 SOH 예측 오차가 1% 이내라는 의미입니다.

**Q: CDE가 왜 -49나 나오냐?**
A: 불규칙 시계열에 이론적으로 유리한 모델이지만, 우리 데이터는 세션 간격이 규칙적이어서 LSTM의 귀납적 편향이 더 맞았습니다. 실험 비교 없이 모델을 선택하면 안 된다는 것을 보여줍니다.

**Q: 배터리 처음엔 천천히 닳다가 나중에 갑자기 빨리 닳는 이유?**
A: SEI 막은 √t로 성장해서 초반엔 빠르고 나중엔 느려집니다. 하지만 SOH 85% 아래로 내려가면 리튬 도금 + 양극 구조 붕괴가 시작되어 지수함수적으로 가속됩니다. 이게 "무릎 효과(Knee Effect)"입니다.

**Q: Peukert, Arrhenius 수치는 어디서 나왔나?**
A: Peukert n=0.621은 Safari 2011, Pelletier 2017의 NCM 셀 실측 피팅값. Arrhenius Ea=31kJ/mol은 Wang 2014, Waldmann 2014 인용.

**Q: 앱에서 보여주는 SOH 곡선이 LSTM 예측값이냐?**
A: 물리 모델(Peukert+Arrhenius+2-phase)로 생성한 곡선에 LSTM의 현재 SOH 추정값으로 보정(앵커링)한 것입니다. 미래 곡선 자체는 물리 법칙 기반이라 데이터 외 구간도 신뢰할 수 있습니다.
